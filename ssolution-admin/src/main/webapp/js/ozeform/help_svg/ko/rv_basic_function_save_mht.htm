MHT 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 Web Page Save File(*.mht)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

글꼴 :

글꼴과 글꼴 크기를 설정합니다.

글꼴 : 라벨의 글꼴을 일괄적으로 변경합니다. 저장 글꼴 사용을 체크한 경우 각 라벨에 설정된 글꼴을 무시하고, 콤보 박스에서 설정한 글꼴이 모든 라벨에 적용됩니다.

크기 : 라벨의 글꼴 크기를 일괄적으로 변경합니다. 글꼴 크기를 설정한 경우 각 라벨에 설정된 글꼴 크기를 무시하고, 콤보 박스에서 설정한 글꼴 크기가 모든 라벨에 적용됩니다.

보호 :

MHT 파일을 열 때 입력받을 암호를 입력합니다. 암호가 설정된 MHT 파일을 열면 암호를 입력받는 웹 페이지가 실행되고 암호가 일치한 경우에만 보고서가 표시됩니다. 단, MHT 파일에 저장된 보고서의 텍스트만 암호화되고, 이미지 데이터는 암호화되지 않습니다.
