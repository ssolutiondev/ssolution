TXT 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 Tab Separated(*.txt)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

구분 :

구분자를 설정합니다.

구분자 : 컬럼 구분자를 설정합니다.

tab : tab

space : space

| : |

/ : /

~ : ~

comma : comma

empty : empty

Other : 그 외 값

사용자 정의 : 기본적으로 제공하는 문자 외에 다른 문자를 구분자로 설정합니다.

페이지 구분 :

페이지 구분을 설정합니다.

구분형태 : 페이지 구분을 나타내는 텍스트를 설정합니다.

None : 페이지 구분자 없이 저장

n <PAGE> : n <PAGE>

n <PAGE> --- : n <PAGE> ---

--- n <PAGE> : --- n <PAGE>

<PAGE> n  : <PAGE> n

<PAGE> n --- : <PAGE> n ---

--- <PAGE> n : --- <PAGE> n

<PAGE> : 페이지 구분형태에서 <PAGE>를 대체할 텍스트를 설정합니다.

페이지 간격 :

페이지 간격을 나타낼 줄 수를 설정합니다. 페이지 구분형태와 페이지 간격 줄 수를 모두 설정한 경우 페이지 구분형태를 먼저 표시하고 그 다음 줄에 지정한 줄 수 만큼의 빈 줄이 표시됩니다.

라인 제거 :

라인 제거를 설정합니다.

라인 제거 : 특정 라인을 제거하여 TXT로 저장하고, 제거할 라인 범위를 설정합니다.

첫 페이지 제외 : 보고서의 첫 페이지만 제외하고 라인을 제거합니다.

숫자문자 :

숫자문자를 설정합니다.

자릿수 구분기호 추가 : 숫자형 텍스트의 경우 1000 단위 구분 기호(,)를 추가합니다.

자릿수 구분기호 제거 : 숫자형 텍스트의 경우 1000 단위 구분 기호(,)를 제거합니다.

저장 방식 :

저장 방식을 설정합니다.

테이블 형식으로 저장 : 테이블, 크로스탭, 고정 테이블의 자식 라벨 중 병합된 라벨이 있을 경우 라벨을 분할하여 저장합니다. 분할된 라벨 중 맨 왼쪽 위에 있는 라벨에만 텍스트를 적용하여 저장하고, 나머지 라벨은 공백 문자로 저장됩니다.

인코딩 : 인코딩 타입을 설정합니다.

ANSI : ANSI

유니코드 : 유니코드
