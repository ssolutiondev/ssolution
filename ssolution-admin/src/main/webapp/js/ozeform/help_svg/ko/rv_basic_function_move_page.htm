페이지 이동

 Top  Previous  Next

 

 

오즈 리포트로 디자인한 보고서의 구조는 단순히 세로 또는 가로 방향으로만 구성되는 것이 아니라 가로 방향과 세로 방향이 조합된 형태 즉, 행/열 매트릭스 구조로 구성됩니다.

페이지를 앞뒤로 이동하거나 특정 페이지로 바로 이동하려면 페이지 이동 툴바를 이용하고, 행/열 매트릭스 구조에서 이동하려면 페이지 선택 창을 이용하시기 바랍니다.

 

 

페이지 이동 툴바로 페이지 이동

페이지를 앞뒤로 이동하거나 특정 페이지로 바로 이동하려면 툴바의 페이지 이동 아이콘에서 원하는 방향을 클릭하거나 페이지 번호를 직접 입력합니다.
