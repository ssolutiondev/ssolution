PDF 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 Adobe PDF File(*.pdf)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

문서 정보 :

문서의 정보를 설정합니다.

제목 : 문서 제목을 설정합니다.

작성자 : 문서 작성자를 설정합니다.

주제 : 문서 주제를 설정합니다.

키워드 : 키워드를 설정합니다.

응용프로그램 : 응용 프로그램을 설정합니다.

열기 보호 :

파일을 열 때 사용할 비밀번호를 설정합니다.

편집 보호 :

파일의 권한 및 암호를 변경할 때 사용할 비밀번호를 설정합니다.

사용자 접근 권한 :

사용자 접근 권한을 설정합니다.

인쇄 허용 : 인쇄할 수 있도록 설정합니다. 인쇄 허용을 체크하지 않고 저장한 PDF 파일을 PDF Reader에서 열면 PDF 파일을 인쇄할 수 없도록 [인쇄] 메뉴와 아이콘이 비활성화됩니다.

내용 복사 : 내용을 복사할 수 있도록 설정합니다. 내용 복사를 체크하지 않고 저장한 PDF 파일을 PDF Reader에서 열면 PDF 파일의 내용을 복사할 수 없도록 [복사] 메뉴와 아이콘이 비활성화됩니다.

변경 허용 : 변경 허용 옵션을 설정합니다. 설정한 변경 허용 옵션은 PDF Reader의 문서 속성 다이얼로그 > 보안 옵션에서 확인할 수 있습니다.

차트를 이미지로 저장 :

차트를 이미지로 변환하여 저장합니다.

차트를 이미지로 저장 옵션은 차트에 표시된 텍스트의 글꼴이나 방향이 뷰어에서 표시된 것과 다르게 익스포트될 경우에만 사용하시기 바랍니다.
