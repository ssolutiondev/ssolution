페이지 표시 옵션 설정

 Top  Previous 

 

 

뷰어에 보고서를 표시하는 방법에는 한 페이지씩 표시하는 방법, 화면 크기에 맞춰 한 페이지씩 표시하는 방법, 연속으로 표시하는 방법, 한 페이지씩 연속으로 표시하는 방법, 너비에 맞춰 한 페이지씩 연속으로 표시하는 방법이 있습니다.

 

 

한 페이지씩 보기

보고서 창을 마우스 오른쪽 버튼으로 클릭 > [한 페이지씩 보기] 메뉴를 클릭합니다.

한 페이지씩 보기인 경우 뷰어 창에 보고서를 한 페이지씩만 표시합니다.

 

 

화면 크기에 맞춰 한 페이지씩 보기

툴바의 화면 크기에 맞춰 한 페이지씩 보기 아이콘을 클릭합니다.

화면 크기에 맞춰 한 페이지씩 보기인 경우 뷰어 창의 크기에 맞춰 보고서를 확대/축소하여 한 페이지씩만 표시합니다.

 

 

연속 보기

보고서 창을 마우스 오른쪽 버튼으로 클릭 > [연속 보기] 메뉴를 클릭합니다.

연속 보기인 경우 뷰어 창에 보고서의 모든 페이지를 세로 및 가로 방향을 유지하여 연속된 페이지로 표시합니다.

 

 

한 페이지씩 연속 보기

보고서 창을 마우스 오른쪽 버튼으로 클릭 > [한 페이지씩 연속 보기] 메뉴를 클릭합니다.

한 페이지씩 연속 보기인 경우 보고서의 모든 페이지를 한 페이지씩 세로 방향의 연속된 페이지로 표시합니다.

 

 

너비에 맞춰 한 페이지씩 연속 보기

툴바의 너비에 맞춰 한 페이지씩 연속 보기 아이콘을 클릭합니다.

너비에 맞춰 한 페이지씩 연속 보기인 경우 뷰어 창의 너비에 맞춰 한 페이지씩 연속된 페이지로 표시합니다.
