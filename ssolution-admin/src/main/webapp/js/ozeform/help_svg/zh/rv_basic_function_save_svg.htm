保存为 SVG 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 Scalable Vector Graphics(*.svg)文件格式后单击 [选项] 按钮后进行设置。

 

保存方式 :

设置保存方式。

单文档保存 : 保存为单文档。如果不选中单文档保存选项时将会分页保存为多个单文档。勾选单文档保存时为便于区分各页每页都会显示边框，上 / 下页面之间添加一定的间隔。此外，水平方向扩展的报表保存为一个文件时，忽略水平方向，只按垂直方向进行罗列。
