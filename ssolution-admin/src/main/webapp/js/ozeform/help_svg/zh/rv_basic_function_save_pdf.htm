保存为 PDF 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 Adobe PDF File(*.pdf)文件格式后单击 [选项] 按钮后进行设置。

 

文档信息 :

设置文档信息。

标题 : 设置文档标题。

作者 : 设置作者。

主题 : 设置主题。

关键字 : 设置关键字。

应用程序 : 设置应用程序。

限制打开 :

设置打开文件的密码。

限制修改 :

设置修改文件权限及打开口令的密码。

文档限制 :

设置用户权限。

允许打印 : 允许打印文档。打开不选中此选项而保存的 PDF 文件时，可看到 PDF Reader 中的 [打印] 菜单和c被禁用。

允许内容复制 : 打开不选中此选项而保存的 PDF 文件时，可看到 PDF Reader 中的 [复制] 菜单和按钮被禁用。

允许更改 : 设置允许更改选项。 所设置的允许更改选项可以到 PDF Reader 的文件属性对话框 > 安全选项中查看。

将图表保存为图片 :

将图表转换为图片保存。

此选项只建议在输出的图表中的文本的字体或方向不同与预览所见时使用。
