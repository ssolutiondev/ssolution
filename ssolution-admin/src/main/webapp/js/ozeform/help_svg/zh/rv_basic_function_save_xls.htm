保存为 XLS 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 Microsoft 97-2003 Excel File(*.xls)文件格式后单击 [选项] 按钮后进行设置。

 

自动调整 :

设置自动调整。

自动调整 : 自动调整页面。

宽度 : 设置宽度。

高度 : 设置高度。

显示 :

显示设置。

网格线 : 是否显示 Excel 的网格线。

文本控制 :

设置文本控制。

缩小字体填充 : 设置是否启用 Excel 的单元格格式中的文本控制选项。

比例 :

设置比例。

缩放比例 : 设置打开 Excel 文档时的缩放比例。

凍結窗格 :

设置冻结窗格。

行 : 设置固定模具的行编号。

列 : 设置固定模具的列名。

保存方式 :

设置保存方式。

保持报表原样，属性 : 尽量保持报表原样和属性。

不含颜色，边框，单元格合并 : 忽略标签的颜色、边框、合并等，只以单纯的表格保存文本。此时，页面之间的空白将会自动被删除。

拆分后删除周边单元格 : 拆分合并的单元格后只保留左上角的单元格。举例，报表中的一个标签拆分为 Excel的四个单元格时，只留下左上角的单元格，删除其他三个单元格。被删除单元格左侧的单元格的内容依次左移。

工作表保存 :

设置保存形式。

一列的页保存为一个工作表 : 一列的页保存到一个工作表。

一页保存为一个工作表 : 一页保存到一个工作表。

先行后列 : 按先行后列的方向生成工作表。

先列后行 : 按先列后行的方向生成工作表。

页面 :

设置页面。

除去页间空白 : 删除页间空隙。

字体 :

设置字号。

字号 : 批量调整标签的字号。忽略各个标签中设定的字号，批量换成在此设定的字号。

線條 :

设置删除行。

删除行 : 指定删除行范围，删除指定范围的行保存。

第一页除外 : 删除第一页除外的其他页中的指定删除行。

只选第一页 : 只删除第一页中的指定删除行。

所有页 : 删除所有页中的指定删除行。

格式 :

设置单元格格式。

把文本型数字转换为数字 : 将文本形数字转换为数字，并设置单元格数字格式。此时，将自动选择分类。例如设定为 #,###.00 时可自动默认为自定义或货币。如果不选中此选项保存时单元格会显示 () 图标。

只适用数值型标签 : 只将设计时将 Formt 设定为数字的标签转为数字。如果要将所有文本形数字转为数字时，取消选中该选项。

转换为常规(单元格格式) : 将单元格格式中的数字分类转换成常规。

单元格最小值 :

设置单元格的最小大小。

宽度 : 设置单元格最小宽度。如果导出的 Excel 单元格宽度小于设定值时，将合并于右侧单元格。

高度 : 设置单元格最小高度。如果导出的 Excel 单元格宽度小于设定值时，将合并于下侧单元格。

保存类型 :

设置保存类型。

以mht格式保存 : 保存为 mht 格式的 Excel。
