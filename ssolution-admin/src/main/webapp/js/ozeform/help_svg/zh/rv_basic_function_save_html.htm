保存为 HTML 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 Web Page(*.html)文件格式后单击 [选项] 按钮后进行设置。

 

设置字号 :

批量调整标签的字号。 忽略各个标签中设定的字号，批量换成在此设定的字号。

页间间隔 :

设置显示分页间隔的行数。

行数 / 水平 : 保存为一个文档时，各页之间的水平行数。

行数 / 垂直 : 保存为一个文档时，各页之间的垂直行数。

偏移 :

设置偏移量。

偏移量 / X轴 : 文档的开始位置的 X轴坐标。

偏移量 / Y轴 : 文档的开始位置的 Y轴坐标。

保存方式 :

设置保存方式。

分页保存 : 分页保存为多个文档。不选中此选项时会保存为单文档。

编码 : 设置编码。

ANSI : ANSI

Unicode : Unicode

UTF-8 : UTF-8

页面 :

设置页面。

标题 : 设置 HTML 标题。

版本 :

设置版本。

HTML 版本 : 设置 HTML 版本。

None : 不指定 HTML 版本。不指定时靠左对齐报表。

4.01 : 指定为 HTML 4.01。指定为 4.01 时报表居中对齐。

限制打开 :

输入打开 HTML 文件时需要输入的口令。打开设置了口令的 HTML 文件时输入密码才能打开网页同时只有口令正确时才能显示报表。但是，HTML 文件之外以图片生成的文件不会加密。
