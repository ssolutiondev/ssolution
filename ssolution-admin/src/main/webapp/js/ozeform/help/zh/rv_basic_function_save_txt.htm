保存为 TXT 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 Tab Separated(*.txt)文件格式后单击 [选项] 按钮后进行设置。

 

分隔符 :

设置分隔符。

分隔符 : 设置分隔符。

tab : tab

space : space

| : |

/ : /

~ : ~

comma : comma

empty : empty

Other : 自定义

自定义 : 自定义分隔符。

分页符 :

设置分页符。

分页符 : 设置分页符。

None : 不插入分页符

n <PAGE> : n <PAGE>

n <PAGE> --- : n <PAGE> ---

--- n <PAGE> : --- n <PAGE>

<PAGE> n  : <PAGE> n

<PAGE> n --- : <PAGE> n ---

--- <PAGE> n : --- <PAGE> n

<PAGE> : 分页符中的 <PAGE>。

页间间距 :

设置页面和页面之间的空白行数。如果同时设置了分页符和页间间隔，将先插入分页符，在分页符下一行开始插入空白行。

删除行 :

设置删除行。

删除行 : 指定删除行范围，删除指定范围的行保存。

第一页除外 : 删除第一页除外的其他页中的指定删除行。

数字 :

设置数字格式。

添加千位分隔符 : 添加千位分隔符。

取消千位分隔符 : 取消千位分隔符。

保存方式 :

设置保存方式。

保存为表格 : 表格、交叉表、固定表格的子标签中有被合并的标签时，将会拆分合并的标签。拆分的标签中只保留左上角的标签的内容，其他显示空白。

编码 : 设置编码。

ANSI : ANSI

Unicode : Unicode
