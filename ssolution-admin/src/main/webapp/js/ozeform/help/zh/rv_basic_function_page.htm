页面显示选项设置

 Top  Previous  Next

 

 

在 OZ Viewer 中视图方法有，单页、按适合画面单页、连续、单页连续、按适合宽度单页连续等。

 

 

单页

右键单击报表窗口 > [页面显示] > 单击 [单页] 菜单。

选择单页时，报表窗口中只会显示一个页面。

 

 

按适合页面单页图标

单击工具栏的按适合页面单页图标。

选择适合画面单页时，窗口中只显示一个页面并将页面缩放至适合窗口。

 

 

连续

右键单击报表窗口 > [页面显示] > 单击 [连续] 菜单。

选择连续时，以报表的行列结构显示报表的所有页面。

 

 

单页连续

右键单击报表窗口 > [页面显示] > 单击 [单页连续] 菜单。

选择单页时，以纵向滚动画面显示报表的所有页。

 

 

按适合页面单页图标

单击工具栏的按适合页面单页图标。

选择按适合宽度单页连续时，以纵向滚动画面显示报表的所有页，并将页面缩放至适合宽度。
