保存数据

 Top  Previous  Next

 

 

单击工具栏的保存数据图标。

保存数据功能是所绑定报表的数据树中所添加的所有数据按照文本格式进行保存的功能。

在弹出的保存数据对话框中设置分隔符、保存位置、编码等选项后单击 [确定] 按钮。

 

分隔标记 :

设置分隔符。

分隔符 : 设置分隔符。

tab : tab

space : space

| : |

/ : /

~ : ~

comma : comma

Other : 自定义

自定义 : 自定义分隔符。

使用缩进功能 : 如果要保存的数据是主从关系 (Master-Detail) 数据时，为了区分 Master 数据和 Detail 数据，缩进 Detail 数据。

设置保存方式。

编码 : 设置编码。

Unicode : Unicode
