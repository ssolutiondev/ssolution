保存为 TIF 文件

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 TIFF Image(*.tif)文件格式后单击 [选项] 按钮后进行设置。

 

编码方式 :

设置编码格式。

G3 : G3 格式。

G4 : G4 格式。

JPG : JPG 格式。

保存方式 :

设置保存方式。

单文档保存 : 保存为单文档。如果不选中单文档保存选项时将会分页保存为多个单文档。

比例 :

按比例缩放 TIF 文件。

缩放比例 : 设置 TIF 文件的缩放比例。

自定义大小 :

自定义 TIF 文件的大小。

宽度 : 设置 TIF 文件的宽度。

高度 : 设置 TIF 文件的高度。

根据分辨率调整大小 :

根据分辨率调整 TIF 文件的大小。

分辨率 : 设置分辨率。
