基本选项

 Top  Previous  Next

 

 

在工具栏中单击保存图标。

在保存对话框中选择 OZ Report Data File(*.ozd)之外的文件格式后单击 [选项] 按钮后进行设置。

 

保存方式 :

设置保存方式。

单页保存 : 将一个多页文档转为单页文档保存。

分页保存 : 将转为单页的多页文档重新分页保存。

多文档 :

设置多文档的保存方式。

单文档保存 : 将多个报表保存为一个文档。

多文档保存 : 将多个报表文件分别保存为各个文档。

保存范围 :

设置保存范围。

全部 : 保存所有页。

当前页 : 仅保存当前画面中的页。

选定页 : 仅保存在页面导航中所选的页。

自定义 : 指定保存页码范围。

组件 :

指定要保存的组件。

保存链接 :

设置是否保存链接。

URL链接 : 保存时保持 URL 链接。

目录链接 : 保存时保持目录链接。

压缩保存 :

设置是否压缩保存。如果输入密码可以将压缩文件加密。
