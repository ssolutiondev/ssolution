设置页边距

 Top  Previous 

 

 

单击工具栏的其他菜单图标 > 单击 [页面设置] 菜单。

在页面设置对话框中设置页边距、度量单位等选项后单击 [确定]。

 

页边距 :

设置页边距。

可设置左、上边距，要调整右、下边距，请选中自动调整选项。

度量单位 :

设置度量单位。

单位 : 设置页边距单位。

pt : 磅

inch : 英寸

cm : 厘米

页边距选项 :

设置页边距选项。

默认页边距 : 使用默认页边距。

自动调整 : 自动调成边距。选中此选项后更改一边的边距时，将自动调整其他方向的边距，报表被自动缩放。
