Save as a MHT file

 Top  Previous  Next

 

 

Click the Save icon on the toolbar.

Select the file format to Web Page Save File(*.mht), click the [Option] button.

 

Font :

Set the font and font size for all text in MHT file to be created. If set, font and font size of all labels in the report are ignored.

Font : Font name.

Size : Font size.

Restriction :

Enter the password that is needed to open the exported MHT files. When you open the password-protected MHT file, the web page for inputting the password will be executed. The report is displayed only when the password is matched. But, the text of the report is encrypted, the image data is not encrypted.
