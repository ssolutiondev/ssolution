Move the page

 Top  Previous  Next

 

 

The pages of OZ Report consists of a matrix of rows and columns. You can navigate to a page in column or row direction.

The Page move toolbar allows to navigate to the next, previous or specified page number. The Page coordinate window allows the user to visually move to the page on matrix of rows and columns.

 

 

Move the page using the Page move toolbar

Click the page move icon on the Page move toolbar. Enter in the text box to the page number you want to move.

 

 

Move the page using the Page coordinate window

Click the Etc icon on the toolbar > Click the [Page coordinate window] menu. Click the Move up, down, left, and right button.
