Edit the Chart

 Top  Previous  Next

 

 

Change the chart type

Right-click on the chart > Click the [Change properties] menu. Click the type of chart in the Edit Chart dialog box and click the [OK] button.

 

 

Move the position

Click the chart. Drag & drop to the position you want.

 

 

Rotate the chart

Right-click on the chart > Click the [Rotatable] menu. Move the mouse in the desired direction while holding down the left mouse button.

 

 

You can edit the chart only designed to be able to edit the chart.
