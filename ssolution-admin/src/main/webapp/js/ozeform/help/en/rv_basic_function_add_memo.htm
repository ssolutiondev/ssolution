Add a memo

 Top  Previous  Next

 

 

Click the Add memo icon on the toolbar, and draw a memo box by drag & drop.

To enter text in memo, double-click on a memo, and then enter text. After the end of the text entry, click any report area other than the edit box.

To set the font, background color, the border and other properties of a memo, click the [Change properties] menu on the pop-up menu of memo.

To change a memo to the image, bar code, HTML labels, etc., click the component type menu.
