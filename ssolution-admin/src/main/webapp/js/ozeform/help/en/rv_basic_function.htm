Basic features

 Top  Next

 

 

You can show and print reports using the OZ Report Viewer. And you can export reports to various file formats(PDF, XLS, DOC, PPT, HTML, MHT, CSV, TXT, JPG, PNG, etc.).

Also, you can add Memo and do simple editing according to your needs and open the OZD file.

 

 

The OZD stands for OZ report Data, which have report forms and data.
