CSVファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコンをクリックします。

保存ダイアログでファイルの種類をComma Separated Values File(*.csv)で選択して[設定]ボタンをクリックして設定します。

 

文字の区切り

区切り文字を設定します。

区切り文字：列の区切り文字を設定します。

tab：tab

space：space

|：|

/：/

~：~

comma：comma

empty：empty

Other：その他

ユーザー定義：基本的に提供する文字以外の文字を区切り文字で設定します。

ページの区切り

ページの区切りを設定します。

区切り形式：ページの区切りを表示するテキストを設定します。

None：ページの区切りなしに保存

n <PAGE>：n <PAGE>

n <PAGE> ---：n <PAGE> ---

--- n <PAGE>：--- n <PAGE>

<PAGE> n：<PAGE> n

<PAGE> n ---：<PAGE> n ---

--- <PAGE> n：--- <PAGE> n

<PAGE>：区切り形式で<PAGE>を代替するテキストを設定します。

ページの間隔

ページの間隔を区別する行数を設定します。ページの区切り形式とページの間隔を全て設定した場合、先にページの区切りを表示して、その次に行に設定した個数分の空白行が表示されます。

行の削除

行削除を設定します。

行の削除：特定行を削除してCSVで保存して削除する行の範囲を設定します。

先頭ページを除外：レポートの先頭ページのみ除外して行を削除します。

数字

数字文字を設定します。

桁数の区切り文字を追加：数値テキストの場合、1000単位区切り記号(,)を追加します。

桁数の区切り文字を削除：数値テキストの場合、1000単位区切り記号(,)を削除します。

保存方式

保存方式を設定します。

テーブル形式で保存：テーブル、クロスタブ、固定テーブルの子ラベルの中で結合されたラベルがある場合、ラベルを分割して保存します。分割されたラベルの中で左側の上にあるラベルにのみテキストを適用して他のラベルは空白文字で保存されます。

エンコード：エンコードタイプを設定します。

ANSI：ANSI

UNICDOE：UNICODE
