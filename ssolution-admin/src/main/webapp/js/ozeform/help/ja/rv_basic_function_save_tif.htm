TIFファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコンをクリックします。

保存ダイアログでファイルの種類をTIF Image(*.tif)で選択して[設定]ボタンをクリックして設定します。

 

エンコード方式

エンコード方式を設定します。

G3：G3方式でエンコードします。

G4：G4方式でエンコードします。

JPG：JPG方式でエンコードします。

保存方式

保存方式を設定します。

１つのファイルに保存：レポートを１つのファイルで保存します。１つのファイルに保存をチェックしないとページ別に別のファイルで保存されます。

比率

TIFファイルのサイズを比率で設定します。

拡大/縮小比率：TIFファイルの拡大/縮小比率を設定します。

サイズ

TIFファイルのサイズを直接設定します。

幅：TIFファイルの幅を設定します。

高さ：TIFファイルの高さを設定します。

解像度によってサイズ決定

TIFファイルのサイズを解像度によって決定します。

DPI：ファイル解像度を設定します。
