OZ Report Viewerの基本機能

 Top  Next

 

 

OZ Report ViewerはOZ Report Designerで作成されたレポートをプレビューして保存、印刷、テキストの検索などができるOZ Report専用Viewerです。

また、OZDファイルをViewerに表示したり、メモを追加するなどユーザー環境に合わせて簡単に編集できます。

 

 

OZDファイルはOZ Report Dataファイルの略字で、レポートとデータを組み合わせたOZ Report専用ファイルです。
