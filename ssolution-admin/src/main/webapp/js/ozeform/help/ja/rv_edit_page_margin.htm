ページ設定

 Top  Previous 

 

 

ツールバーのその他のメニューアイコンをクリック　＞　[ページ設定]メニューをクリックします。

ページ設定ダイアログで余白、使用単位などを設定した後、[OK]ボタンをクリックします。

 

余白

ページの余白を設定します。

左側、上側余白が設定できます。右側、下側余白を設定するためには自動調整をチェックしてください。

使用単位

使用単位を設定します。

単位：ページ余白設定時に使用する単位を設定します。

pt：ポイント

inch：インチ

cm：センチメートル

余白オプション

ページの余白オプションを設定します。

基本余白使用：レポートの余白を既定値で設定します。

自動調整：レポートの余白を自動的に調整します。自動調整をチェックして余白を変更するとレポートが自動的に拡大または縮小され、特定方向の余白によって他の余白が自動的に調節されます。
