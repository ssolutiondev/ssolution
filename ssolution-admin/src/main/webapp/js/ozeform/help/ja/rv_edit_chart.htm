チャート編集

 Top  Previous  Next

 

 

チャートタイプ変更

チャートのタイプを編集するためにはチャートを選択して右クリック　＞　[属性変更]メニューをクリック　＞　チャートの編集ダイアログで任意のチャートのタイプを選択した後[OK]ボタンをクリックします。

 

 

位置移動

チャートを移動するためにはチャートをクリックした後、任意の位置にドラッグ&ドロップします。

 

 

チャートの回転

チャートを回転するためにはチャートを選択して右クリック　＞　[回転可能]メニューをクリックした後、左クリックした状態で任意の方向にマウスを移動します。

 

 

OZ Viewer開発者がチャートの編集、位置などが変更できるように、レポートをデザインした場合にのみチャートのテキストが編集できます。
