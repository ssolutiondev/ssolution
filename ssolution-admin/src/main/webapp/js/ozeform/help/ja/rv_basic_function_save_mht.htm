MHTファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコンをクリックします。

保存ダイアログでファイルの種類をWeb Page Save File(*.mht)で選択して[設定]ボタンをクリックして設定します。

 

保存方式

フォント名とフォントサイズを設定します。

フォント：ラベルのフォントを一括して変更します。保存フォントを使用をチェックした場合、各ラベルに設定されたフォントを無視してコンボボックス内で設定したフォントが全てのラベルに適用されます。

サイズ：ラベルのフォントサイズを一括して変更します。フォントサイズを設定した場合、各ラベルに設定されたフォントサイズを無視してコンボボックス内で設定したフォントサイズが全べてのラベルに適用されます。

保護

MHTファイルを開くとき、入力されるパスワードを設定します。パスワードが設定されたMHTファイルを開くと、パスワードの入力を貰えるWebページが実行され、パスワードが一致した場合にのみ、レポートが表示されます。ただし、MHTファイルに保存されたレポートのテキストのみ、暗号化されて、イメージデータは暗号化されません。
