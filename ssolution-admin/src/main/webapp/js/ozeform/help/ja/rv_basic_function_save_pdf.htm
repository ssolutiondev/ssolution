PDFファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコンをクリックします。

保存ダイアログでファイルの種類をAdobe PDF File(*.pdf)で選択して[設定]ボタンをクリックして設定します。

 

文書情報

文書の情報を設定します。

タイトル：レポートのタイトルを入力します。

作成者：レポートの作成者を入力します。

サブタイトル：レポートのサブタイトルを入力します。

キーワード：レポート内部で使用するキーワードを入力します。

アプリケーション：アプリケーションを入力します。

読み取りパスワード

エクスポートされたPDF文書の保護のために文書を開く時に使用するパスワードを設定します。

書き込みパスワード

権限及びパスワードの変更時に使用するパスワードを設定します。

ユーザーのアクセス権限

ユーザーのアクセス権限を設定します。

印刷可能：印刷できるように設定します。印刷可能をチェックせずに保存したPDFファイルをPDF Readerで開くと、PDFファイルが印刷できないように[印刷]メニューとアイコンは非アクティブ化されます。

内容のコピー：内容がコピーできるように設定します。内容コピーをチェックせずに保存したPDFファイルをPDF　Readerで開くと、PDFファイルの内容がコピーできないように[コピー]メニューとアイコンは非アクティブ化されます。

変更を許可：変更許可のオプションを設定します。設定された変更許可のオプションはPDF Readerの文書属性ダイアログ > セキュリティのオプションから確認できます。

チャートをイメージで保存

チャートをイメージに変換して保存します。

チャートをイメージで保存オプションはチャートに表示されたテキストのフォントまたは方向がViewerに表示された結果と異なってエクスポートされる場合にのみ使用してください。
