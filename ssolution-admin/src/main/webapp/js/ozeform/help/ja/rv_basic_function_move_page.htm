ページ移動

 Top  Previous  Next

 

 

OZ Reportでデザインしたレポートの構造は単純に縦方向または横方向で構成されることではなく、横方向と縦方向を組み合わせた形式、行/列マトリックス構造で構成されます。

ページを上下に移動したり特定ページに即時移動するためにはページ移動ツールバーを使用して、行/列マトリックス構造で移動するためにはページ選択ダイアログを使用してください。

 

 

ページ移動ツールバーでページ移動

ページを前後に移動したり特定ページに即時移動するためには、ツールバーの番号入力及び前後移動アイコンで任意の方向をクリックしたりページ番号を直接入力します。

 

 

ページ選択ダイアログでページ移動

ツールバーのその他のメニューアイコンをクリック　＞　[ページ選択]メニューをクリック　＞　上下左右ページ移動ボタンをクリックします。
