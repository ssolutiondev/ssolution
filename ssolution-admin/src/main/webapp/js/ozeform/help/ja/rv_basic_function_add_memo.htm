メモ追加

 Top  Previous  Next

 

 

ツールバーのメモ追加アイコンをクリックしてレポートウィンドウで任意の位置にドラッグ&ドロップしてメモを描画します。

追加されたメモにテキストを入力するためにはメモをダブルクリックしてテキスト入力後、テキストエディター以外のレポート領域をクリックして適用します。フォント、塗りつぶし色、枠線などはメモを右クリック　＞　[属性変更]メニューをクリック　＞　ラベル書式ダイアログで変更可能です。

追加されたメモをイメージ、バーコード、HTMLラベルなどに変更するためは、メモを右クリック ＞ コンポーネントタイプのメニューをクリックします。
