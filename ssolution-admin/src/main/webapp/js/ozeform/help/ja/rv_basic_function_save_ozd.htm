OZDファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコンをクリックします。

保存ダイアログでファイルの種類をOZ Report Data File(*.ozd)で選択して[設定]ボタンをクリックして設定します。

 

パスワード

保存されるファイルにパスワードを設定します。

パスワードの確認

入力したパスワードを確認します。

編集内容を含む

プレビュー画面で追加したメモ、コメント、吹き出し図形（線、矢印、四角形、円）、テキスト編集内容をOZDファイルに含めて保存します。

複数レポートの保存

現在、開いている全てのレポートを１つのOZDファイルで保存します。

オプションをチェックしないと現在表示されているレポートのみ保存されます。

イメージを含む

イメージのパスがURL形式(http://、ozp://)で設定された場合、該当イメージをOZDファイルに含めて保存します。イメージを含むオプションはイメージだけでなくてテキストを特定URL(http://、ozp://)を通じて取得するように設定したノートにも適用されます。
