페이지 여백 설정

 Top  Previous 

 

 

툴바의 다른 메뉴 아이콘 클릭 > [페이지 설정] 메뉴를 클릭합니다.

페이지 설정 다이얼로그에서 여백, 사용 단위계 등을 설정한 후 [확인] 버튼을 클릭합니다.

 

여백 :

여백을 설정합니다.

왼쪽, 위쪽 여백을 설정할 수 있으며, 오른쪽, 아래쪽 여백을 설정하려면 자동 조정을 체크하시기 바랍니다.

사용 단위계 :

사용 단위계를 설정합니다.

단위 : 페이지 여백 설정 시 사용할 단위를 설정합니다.

pt : 포인트

inch : 인치

cm : 센티미터

여백 옵션 :

여백 옵션을 설정합니다.

기본 여백 사용 : 보고서 여백을 기본 값으로 설정합니다.

자동 조정 : 보고서의 여백을 자동으로 조정합니다. 자동 조정을 체크하고 여백을 변경하면 보고서가 자동으로 확대 또는 축소되며, 특정 방향의 여백에 따라 다른 방향의 여백이 자동으로 조절됩니다.
