메모 추가

 Top  Previous  Next

 

 

툴바의 메모 추가 아이콘을 클릭한 후 보고서 창의 원하는 위치에 드래그 & 드롭하여 메모를 그립니다.

추가된 메모에 텍스트를 입력하려면 메모를 더블 클릭하고, 텍스트 입력 후 편집 창 외의 보고서 영역을 클릭하여 적용합니다. 글꼴, 바탕색, 테두리 등은 메모를 마우스 오른쪽 버튼으로 클릭 > [속성 변환] 메뉴 클릭 > 라벨 서식 다이얼로그에서 변경할 수 있습니다.

추가된 메모를 이미지, 바코드, HTML 라벨 등으로 변경하려면 메모를 마우스 오른쪽 버튼으로 클릭 > 컴포넌트 타입 메뉴를 클릭합니다.
