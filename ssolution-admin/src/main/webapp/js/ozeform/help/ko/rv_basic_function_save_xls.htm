XLS 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 Microsoft 97-2003 Excel File(*.xls)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

자동 맞춤 :

자동 맞춤을 설정합니다.

자동 맞춤 : 페이지 배율을 자동 맞춤으로 설정합니다.

너비 : 너비를 설정합니다.

높이 : 높이를 설정합니다.

창 옵션 :

창 옵션을 설정합니다.

눈금선 : 엑셀의 눈금선 표시 여부를 설정합니다.

텍스트 조정 :

텍스트 조정 옵션을 설정합니다.

셀에 맞춤 : 엑셀의 셀 서식 중 셀에 맞춤 여부를 설정합니다.

배율 :

배율을 설정합니다.

확대/축소 비율 : 엑셀 파일을 열 때 확대/축소 배율을 설정합니다.

틀 고정 :

틀 고정을 설정합니다.

행 : 틀 고정할 행 번호를 설정합니다.

열 : 틀 고정할 열 이름을 설정합니다.

저장 방식 :

저장 방식을 설정합니다.

폼 형태, 속성 유지 : 보고서 형태와 속성을 최대한 유지하면서 저장합니다.

색상, 외곽라인, 셀합치기 무시 : 라벨의 색, 테두리, 병합 상태 등을 무시하고, 단순 리스트 형태로 라벨의 텍스트만 저장합니다. 이때 페이지 사이 공백은 자동으로 제거됩니다.

분할 후 주변 셀 제거 : 병합된 라벨을 분할한 후 주변 셀을 제거합니다. 예를 들어, 보고서에서는 하나의 라벨이 엑셀에서는 네 개의 셀로 분할된 경우 셀 위치를 기준으로 맨 왼쪽 위에 있는 한 개의 셀만 남기고 나머지 셀을 제거합니다. 셀이 제거된 후에는 제거된 셀의 오른쪽에 있는 텍스트가 제거된 셀 위치로 이동됩니다.

저장 형태 :

저장 형태를 설정합니다.

페이지 열별 시트 저장 : 같은 열에 있는 페이지를 같은 시트에 저장합니다.

페이지별 시트 저장 : 페이지별로 시트를 구분하여 저장합니다.

Column 우선 : 열 방향 우선으로 시트를 생성합니다.

Row 우선 : 행 방향 우선으로 시트를 생성합니다.

페이지 :

페이지를 설정합니다.

페이지 사이 공백 제거 : 페이지 사이의 공백을 제거합니다.

글꼴 :

글꼴 크기를 설정합니다.

크기 : 라벨의 글꼴 크기를 일괄적으로 변경합니다. 글꼴 크기를 설정한 경우 각 라벨에 설정된 글꼴 크기를 무시하고, 콤보 박스에서 설정한 글꼴 크기가 모든 라벨에 적용됩니다.

라인 :

라인을 설정합니다.

라인 제거 : 특정 라인을 제거하여 엑셀로 저장하고, 제거할 라인 범위를 설정합니다.

첫 페이지 제외 : 보고서의 첫 페이지만 제외하고 라인을 제거합니다.

첫 페이지만 : 보고서의 첫 페이지만 라인을 제거합니다.

모든 페이지 : 보고서의 모든 페이지 라인을 제거합니다.

서식 :

셀 서식을 설정합니다.

숫자형 텍스트를 숫자로 변환 : 숫자형 텍스트를 숫자로 변환하고, 변환되는 셀의 표시 형식을 설정합니다. 텍스트 박스에 입력한 표시 형식은 엑셀의 셀 표시 형식에 적용됩니다. 이때 셀 서식 중 범주는 엑셀 파일을 열기한 엑셀 프로그램이 인식하는 범주로 자동 선택됩니다. 예를 들어, #,###.00으로 설정한 경우 범주가 사용자 지정으로 선택될 수도 있고, 통화로도 선택될 수 있습니다. 숫자형 텍스트를 숫자로 변환을 체크하지 않고 저장한 엑셀을 열면 숫자 텍스트가 입력된 셀은 오류 검사 아이콘()이 표시됩니다.

표시 형식이 숫자인 라벨만 적용 : 보고서를 디자인할 때 설정한 라벨의 표시 형식이 숫자인 라벨만 숫자로 변환합니다. 보고서에 표시된 모든 숫자형 텍스트를 숫자로 변환하려면 표시 형식이 숫자인 라벨만 적용을 체크 해지하시기 바랍니다.

일반(셀표시 형식)으로 변환 : 셀 표시 형식을 일반으로 변환합니다.

셀 최소 크기 :

셀 최소 크기를 설정합니다.

너비 : 셀 최소 너비를 설정합니다. 엑셀로 저장된 셀의 너비가 설정한 값보다 작을 경우 오른쪽 셀에 병합됩니다.

높이 : 셀 최소 높이를 설정합니다. 엑셀로 저장된 셀의 높이가 설정한 값보다 작을 경우 아래쪽 셀에 병합됩니다.

저장 형식 :

저장 형식을 설정합니다.

mht형식으로 저장 : mht 형식의 엑셀로 저장합니다.
