OZD 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 OZ Report Data File(*.ozd)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

암호 :

파일의 비밀번호를 설정합니다.

확인 :

비밀번호를 한번 더 확인합니다.

편집 내용 포함 :

미리보기 화면에서 추가한 메모, 주석, 콜아웃 도형(선, 화살표, 사각형, 원), 텍스트 편집 내용을 OZD 파일에 포함하여 저장합니다.

모든 보고서 저장 :

현재 열린 모든 보고서를 하나의 OZD 파일로 저장합니다.

옵션을 체크하지 않으면 현재 보고서 창에 표시된 보고서만 저장됩니다.

이미지 포함 :

이미지 경로가 URL 형태(http://, ozp://)로 설정된 경우 해당 이미지를 OZD 파일에 포함하여 저장합니다.

이미지 포함 옵션은 이미지뿐만 아니라 텍스트를 특정 URL(http://, ozp://)을 통해 가져오도록 설정한 노트에도 적용됩니다.
