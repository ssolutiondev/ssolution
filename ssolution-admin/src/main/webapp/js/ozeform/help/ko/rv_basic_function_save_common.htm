기본 옵션 설정

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 OZ Report Data File(*.ozd)을 제외한 형식으로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

저장 방식 :

보고서 저장 방식을 설정합니다.

한페이지로 저장 : 여러 페이지의 보고서를 한 페이지로 바인딩하여 저장합니다.

페이지별로 저장 : 여러 페이지의 보고서가 한 페이지로 바인딩된 경우 페이지별로 다시 바인딩하여 저장합니다.

다중 문서 :

다중 문서 저장 옵션을 설정합니다.

한파일로 저장 : 여러 개의 보고서를 하나의 파일로 저장합니다.

모든 문서 저장 : 여러 개의 보고서를 각각의 파일로 저장합니다.

저장 영역 :

저장할 페이지 영역을 설정합니다.

전체 페이지 : 보고서의 모든 페이지를 저장합니다.

현재 페이지 : 현재 화면에 표시된 페이지만 저장합니다.

선택된 페이지 : 페이지 선택 창의 보고서 구조보기에서 선택한 페이지만 저장합니다.

페이지 지정 : 페이지 범위를 지정하여 저장합니다.

컴포넌트 :

저장할 컴포넌트를 설정합니다.

링크정보 저장 :

링크 정보 저장 옵션을 설정합니다.

URL 링크 : URL 링크 정보를 포함하여 저장합니다.

목차 링크 : 목차 라벨 정보를 포함하여 저장합니다.

압축 저장 :

압축 저장 여부를 설정합니다. 암호를 입력하여 압축 파일을 암호로 보호할 수 있습니다.
