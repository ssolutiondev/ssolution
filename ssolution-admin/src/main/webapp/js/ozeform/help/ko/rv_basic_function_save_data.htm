데이터 저장

 Top  Previous  Next

 

 

툴바의 데이터 저장 아이콘을 클릭합니다.

데이터 저장 기능은 바인딩된 보고서의 데이터 트리에 추가된 모든 데이터를 텍스트 형식으로 저장하는 기능입니다.

데이터 저장 다이얼로그에서 구분자, 인코딩 등을 설정한 후 [확인] 버튼을 클릭합니다.

 

저장 형태 :

구분자를 설정합니다.

구분자 : 컬럼 구분자를 설정합니다.

tab : tab

space : space

| : |

/ : /

~ : ~

comma : comma

Other : 그 외 값

사용자 정의 : 기본적으로 제공하는 문자 외에 다른 문자를 구분자로 설정합니다.

들여쓰기 기능 사용 : 저장할 데이터 모듈이 마스터-디테일 관계로 설정된 경우 마스터 데이터와 디테일 데이터를 구분하기 위해 디테일 데이터를 들여쓰기하여 저장합니다.

저장 방식 :

저장 방식을 설정합니다.

인코딩 : 인코딩 타입을 설정합니다.

ANSI : ANSI

유니코드 : 유니코드
