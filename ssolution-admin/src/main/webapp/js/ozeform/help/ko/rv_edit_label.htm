라벨 편집

 Top  Previous  Next

 

 

텍스트 변경

라벨의 텍스트를 변경하려면 라벨을 더블클릭 > 텍스트 편집 창에서 텍스트를 입력합니다. 입력한 텍스트를 적용하려면 편집 창 외의 보고서 영역을 클릭합니다.

 

 

크기 조절

라벨의 크기를 조절하려면 라벨을 클릭한 후 원하는 크기만큼 드래그 & 드롭합니다.

 

 

위치 이동

라벨의 위치를 이동하려면 라벨을 클릭한 후 원하는 위치로 드래그 & 드롭합니다.

 

 

속성 변경

라벨의 속성을 변경하려면 라벨을 마우스 오른쪽 버튼으로 클릭 > [속성 변환] 메뉴 클릭 > 라벨 서식 다이얼로그에서 단락, 글꼴, 테두리 등의 속성을 변경한 후 [확인] 버튼을 클릭합니다.

 

 

그리기 형태 변경

라벨의 그리기 형태를 변경하려면 라벨을 마우스 오른쪽 버튼으로 클릭 > 원하는 그리기 형태 메뉴를 클릭합니다.

 

 

인쇄 여부 설정

라벨의 인쇄 여부를 설정하려면 라벨을 마우스 오른쪽 버튼으로 클릭 > [인쇄하지 않음] 메뉴를 클릭합니다.

[인쇄하지 않음] 메뉴를 클릭하면 메뉴 앞에 체크 표시가 되며, 해당 라벨은 인쇄되지 않습니다. 다시 라벨을 인쇄하려면 [인쇄하지 않음] 메뉴를 한번 더 클릭합니다.

 

 

오즈 뷰어 개발자가 라벨의 텍스트, 위치, 속성 등을 변경할 수 있도록 보고서를 디자인한 경우에만 라벨을 편집할 수 있습니다.
