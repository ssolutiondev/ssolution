차트 편집

 Top  Previous  Next

 

 

차트 타입 변경

차트의 타입을 변경하려면 차트를 마우스 오른쪽 버튼으로 클릭 > [속성 변환] 메뉴 클릭 > 차트 편집 다이얼로그에서 원하는 차트 타입을 선택한 후 [확인] 버튼을 클릭합니다.

 

 

위치 이동

차트의 위치를 이동하려면 차트를 클릭한 후 원하는 위치로 드래그 & 드롭합니다.

 

 

차트 회전

차트를 회전하려면 차트를 마우스 오른쪽 버튼으로 클릭 > [회전 가능] 메뉴 클릭한 후 마우스 왼쪽 버튼을 누른 상태에서 원하는 방향으로 마우스를 움직입니다.

 

 

오즈 뷰어 개발자가 차트의 타입, 위치 등을 변경할 수 있도록 보고서를 디자인한 경우에만 차트를 편집할 수 있습니다.
