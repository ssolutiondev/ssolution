TIF 파일로 저장

 Top  Previous  Next

 

 

툴바에서 저장 아이콘을 클릭합니다.

저장 다이얼로그에서 파일 형식을 TIFF Image(*.tif)로 선택하고 [옵션] 버튼을 클릭하여 설정합니다.

 

인코딩 방식 :

인코딩 방식을 설정합니다.

G3 : G3 방식으로 인코딩합니다.

G4 : G4 방식으로 인코딩합니다.

JPG : JPG 방식으로 인코딩합니다.

저장 방식 :

저장 방식을 설정합니다.

한 파일에 저장 : 보고서를 하나의 파일로 저장합니다. 한 파일에 저장을 체크하지 않으면 페이지별로 별도의 파일로 저장됩니다.

비율 :

TIF 파일의 크기를 비율로 설정합니다.

확대/축소 비율 : TIF 파일의 확대/축소 비율을 설정합니다.

크기 :

TIF 파일의 크기를 직접 설정합니다.

너비 : TIF 파일의 너비를 설정합니다.

높이 : TIF 파일의 높이를 설정합니다.

해상도에 따라 크기 결정 :

TIF 파일의 크기를 해상도에 따라 결정합니다.

해상도 : 해상도를 설정합니다.
